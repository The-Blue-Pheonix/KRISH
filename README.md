# Smart Agri System

Project scaffold for a smart agriculture platform with:
- FastAPI backend
- React frontend
- ESP8266 IoT firmware

## Quick Start

### Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. `uvicorn app.main:app --reload`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`
